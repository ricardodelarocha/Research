# Computer Science Curricula from Public Institutions

## About this repository
This repository gathers the files of **Segurança Computacional nos Currículos dos Cursos de Ciência da Computação em Instituições de Ensino Público da Região Sul do Brasil** (**_Cybersecurity in Curricula of Computer Science Programs in Public Educational Institutions in Southern Brazil_**) research.
